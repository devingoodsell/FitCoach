# E11 — Diet & Nutrition Guidance (Lightweight)

**Goal:** Simple, preference-aware daily targets and guidance tied to the day's training. **No full
food logging in MVP.** **Wave 1.** Depends on: E2 (diet prefs — E2-PR6), E3. Parallel with: E4, E7, E9, E13.

> Start after E2's diet-preference field lands (or stub the field).

Stories: E11-S1 daily nutrition targets, E11-S2 preference-aware guidance, E11-S3 `[P1]` guidance
tied to today's workout.

## PRs

### E11-PR1 — Nutrition target computation `[backend]` `[contract]` — *E11-S1*
- `internal/diet`: compute calorie + protein target ranges from age, sex, weight, goal weights, and
  the day's training load. Framed as guidance with a **non-medical-advice disclaimer** (E13). API in `backend/api/`.
- Tests: representative profiles → expected target ranges; disclaimer attached.
- Done: deterministic daily targets.

### E11-PR2 — Preference-aware guidance `[backend]` — *E11-S2*
- Tailor suggestions to the dietary pattern (e.g. plant-based protein for vegan); **never violate a
  stated hard dietary constraint**.
- Tests: vegan profile yields no animal-protein suggestion; hard constraints honored.
- Done: guidance respects how the user eats.

### E11-PR3 — Post-workout guidance `[backend]` `[P1]` — *E11-S3*
- After a session, a brief plain-language note for the rest of the day given today's load (e.g.
  emphasize protein on a heavy day); respects diet prefs; non-prescriptive.
- Tests: heavy vs. light day yields appropriately different notes; prefs respected.
- Done: nutrition note ties to today's work.

### E11-PR4 — Diet UI `[android]` — *E11-S1, E11-S2, E11-S3*
- `feature/diet`: show daily targets + guidance with disclaimer; post-session note surface.
- Tests: targets/guidance render; disclaimer present.
- Done: user sees preference-aware targets and a post-workout note.

## Definition of done (epic)
Daily calorie/protein targets compute from profile + load, guidance respects dietary pattern and hard
constraints, an optional post-workout note ties to the day's session, all framed as non-medical guidance.

# FitCoach — AI Health Coach

An adaptive strength & longevity coach for Android. It programs, adjusts, and explains
training around the user's age, injuries, recovery, schedule, and diet — generated by an
LLM that reasons over durable **Coach Memory** and guarded by a deterministic safety layer.

## Monorepo layout

| Path        | What lives here |
|-------------|-----------------|
| `/android`  | Native Android app — Kotlin + Jetpack Compose, first-class Health Connect. |
| `/backend`  | Go service + MySQL. Holds the Claude API key, assembles Coach Memory into prompts, calls Claude, runs the safety layer, persists data. |
| `/docs`     | Product spec, user stories/epics, and per-epic PR plans (`docs/epics/`). |

## Stack (locked — do not change without asking the owner)

- **Client:** Android native, Kotlin + Jetpack Compose, Health Connect.
- **Backend:** Go + MySQL. The Claude/Anthropic API key is **server-side only** — never in the client or the repo.
